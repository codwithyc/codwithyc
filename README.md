기존 10~15분 걸리던 문제 생성 시간을 5분 이내로 단축하고, 실패율을 3 ~ 4% 수준으로 낮췄습니다.

또한 생활기록부 업로드 퍼널에서 1,680명 중 381명의 결제 전환, 전환율 22.68%를 기록한  
핵심 사용자 흐름을 운영하며, KEK/DEK 분리, AES-GCM, TSID, 1년 단위 보관·폐기 정책을 통해  
민감 교육 데이터를 다루는 서비스의 보안 기준을 함께 설계했습니다.
